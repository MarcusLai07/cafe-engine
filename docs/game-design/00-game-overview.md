# Cafe Simulator - Game Overview

## Concept

A cozy isometric pixel art cafe management game where players build and run their own cafe, serve customers, and grow their business through progression and unlocks.

## Genre

- Business Simulation
- Idle/Management
- Casual

## Target Audience

- Players who enjoy relaxing management games
- Fans of pixel art aesthetics
- Ages 10+ (family-friendly)

## Core Experience

**Fantasy:** Own and operate a successful cafe, watching it grow from a small shop to a thriving business.

**Core Loop:**
```
Open Shop → Customers Arrive → Serve Orders → Earn Money → Unlock Items → Improve Cafe → Repeat
```

**Session Length:** 15-30 minutes per session (designed for short play sessions)

## Key Pillars

### 1. Satisfying Progression
- Start small, grow big
- Regular unlocks keep players engaged
- Clear goals to work toward

### 2. Cozy Atmosphere
- Relaxing pixel art visuals
- Pleasant background music
- No fail states or pressure

### 3. Simple but Deep
- Easy to understand core mechanics
- Optional depth through optimization
- No overwhelming complexity

## Game Features

### MVP Features (Phase 6)
- Time system with day/night cycle
- Customer spawning and behavior
- Order system with menu items
- Money and profit tracking
- XP and level progression
- Basic unlocks (new menu items)
- Save/load game state

### Extended Features (Phase 8)
- Staff hiring and management
- Furniture placement and decoration
- Multiple customer types
- Special events

### Future Possibilities
- Multiple cafe locations
- Seasonal events
- Customer relationships
- Achievements/leaderboards

## Visual Style

**Art Style:** Isometric pixel art
- Resolution: 320x180 internal, scaled to display
- Tile size: 32x16 (isometric)
- Character size: ~32x48 pixels
- Color palette: Warm, inviting tones

**Reference Games:**
- Stardew Valley (pixel art, cozy)
- Game Dev Story (business sim, progression)
- Coffee Talk (cafe atmosphere)

## Audio Design

**Music:**
- Calm, jazzy background tracks
- Different themes for different times of day

**Sound Effects:**
- Cash register cha-ching
- Coffee brewing sounds
- Door bell when customers enter
- UI feedback sounds
